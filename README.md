# Sistema HARNESS-SDD

Sistema de micro-agentes SEO para cualquier IA.

---

## 🗂️ Qué hay en este repositorio

Este repositorio tiene dos capas independientes:

### 1. harnesses/ — El cerebro
Los agentes especializados (arneses), uno por módulo de SEO.
El estado de construcción de esta capa se controla en CONTROL.md.
Empieza ahí si la tarea es crear o probar un arnés nuevo.

### 2. proyectos/ — El trabajo real
El registro de proyectos o clientes reales que usan los arneses
ya construidos. El índice de esta capa está en proyectos/PROYECTOS.md.
Empieza ahí si la tarea es sobre un proyecto o cliente concreto,
no sobre construir un arnés.

---

## 🔁 Cómo retomar en cualquier sesión

- ¿Vas a construir o probar un arnés? → lee CONTROL.md primero.
- ¿Vas a trabajar en un proyecto real? → lee proyectos/PROYECTOS.md
  primero, y dentro de ahí el PROYECTO.md del proyecto específico.

Ambos archivos están diseñados para que cualquier IA (Claude,
ChatGPT, Gemini, etc.) entienda el estado actual sin que tengas
que explicar nada desde cero.
