## 📚 CONOCIMIENTO CLAVE DEL MÓDULO

### Definición
- SEO On Page = factores INTERNOS de la web que afectan al
  posicionamiento. También llamado On Site.
- SEO Off Page = factores EXTERNOS (lo que otras webs hacen
  respecto a la tuya). También llamado Off Site.

### Analogía clave
- On Page = lo que TÚ eres como persona en la sociedad
  (tienes control total).
- Off Page = lo que LOS DEMÁS dicen de ti
  (puedes influir, pero no controlar).

### Importancia relativa
- Criterio orientativo: 70% On Page / 30% Off Page.
- No hay documentación oficial — es criterio profesional.
- En mercados muy competidos, el Off Page puede ser decisivo
  para romper el techo de posicionamiento.
- En mercados menos competidos, una excelente optimización
  On Page puede ser suficiente para posicionar.

### Factores y asignaturas On Page
Todos los elementos que están DENTRO de la web:
1. Arquitectura SEO (tratada antes como módulo propio)
2. URLs
3. Headings
4. Contenidos
5. Metadatos
6. Imágenes
7. Enlazado Interno
8. Robots
9. Sitemap
10. Datos Estructurados / Rich Snippets
11. Indexación
12. Canibalizaciones
13. Mobile
14. WPO (velocidad de carga: LCP, FID, CLS)
15. Blog

### Factores Off Page (referencia)
- Perfil de enlaces
- Link Building
- Link Baiting
- Señales Off Page

### Criterio profesional clave
Antes de actuar, analiza:
- ¿En qué nivel de competencia está el mercado?
- ¿Qué factores On Page están sin optimizar?
- ¿Hay un techo de posicionamiento que solo el Off Page puede romper?
- El conocimiento del máster debe dar CRITERIO, no recetas fijas.
