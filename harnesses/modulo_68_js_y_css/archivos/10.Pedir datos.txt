Hola, bienvenidos al noveno punto del curso de JavaScript del Master de sobrevixeo, en el que veremos cómo le podemos pedir al usuario que nos introduzca datos y tratarlos desde nuestro código JavaScript. La primera forma que veremos para solicitar datos a los usuarios es a partir de una ventana emergente. El comando que hace que esto sea posible es el que vemos aquí en el ejemplo, window.prom. Aquí introducimos el mensaje que queremos que el usuario vea y lo que el usuario introduzca en el cuadradito de esta ventana emergente será el valor que obtenga la variable respuesta. Respuesta será un texto aunque el usuario introduzca un número. Es importante saber que todo lo que venga de windows.prom se guardará como texto aunque sea un número, luego lo podemos convertir al número como hemos visto anteriormente si lo que nos interesa es hacer una suma o cualquier otra operación. Si el usuario no escribe respuesta será una cadena vacía, si en esta ventanita que aparece el usuario pulsa en cancelar, la respuesta será igual a nul, es diferente ser nul que ser una cadena vacía. Y comentar que esta opción es la menos recomendable, es poco usable y al igual que nos pasaba con el alert detiene la ejecución de la página y del código de JavaScript hasta que el usuario acepta. Por otro lado vemos la segunda forma de pedir datos al usuario que es a través de campos inputs o de formularios. Con un mente no hay un campo input suelto, hay un formulario que sigue una serie de validaciones. Pero en este ejemplo lo que tenemos es un campo input, un botón y un div de resultado y al final si analizamos el código vemos que el input tiene un id que se entrada, vemos que el botón tiene un evento, un evento que se ejecuta cuando el usuario hace click encima del botón y que este evento lo que hace es ejecutar una función. Esta función se llama el input, la vemos declarada en el general de la página y lo que está haciendo es guardar en una variable que se llama texto el valor que hay dentro del input que con el id entrada y este mismo valor sin alterarlo o modificarle nacer absolutamente nada lo está introduciendo dentro del div que tiene el id resultado. Es un ejemplo simple de lectura de dato pero que bueno lo podéis probar podéis escribir este código y ejecutarlo y veréis que funciona. Y bueno estas son las dos formas principales de solicitar datos al usuario, con esto finalizamos ya el punto de cómo solicitar datos desde JavaScript y pasaremos al último punto al punto 10 donde hablaremos de los principales eventos de este lenguaje.
