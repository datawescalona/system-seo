En este punto hablaremos de los diferentes tipos de variables que hay en este lenguaje y veremos algunos ejemplos de utilización de los mismos. Aunque no es necesario especificar el tipo de datos que almacenar a una variable en JavaScript como hemos comentado anteriormente, JavaScript ofrece una serie de tipos de variables predefinidos. El tipo de variable en Amber permite almacenar números enteros o decimales separados por puntos. Si escribimos el número entre comillas se interpretará como un tipo de variable string, lo que es igual a una cadena de texto. Es el único tipo de variable numérica que dispone JavaScript, otros lenguajes tienen más tipos de variables numéricas, pero en JavaScript únicamente tenemos este tipo. Por otro lado tenemos las variables string que permiten almacenar un texto. El texto se escribí entre comillas, ya sean comillas simples o dobles, y se pueden concatenar con el símbolo más. En el caso de que tengamos dos variables string y simulemos a hacer una suma, lo que hará el lenguaje de este programación será concatenar estas dos palabras y juntarlas. Cualquier tipo de variable que queramos concatenar con un string lo que hará será funcionar como si esta variable que queremos concatenar sea del tipo que sea, trabajará como si fuese un string en esta operación y se unirán, se fusionarán estos dos textos. Vamos a analizar un ejemplo de variables de JavaScript donde veremos un documental HTML en el que incorporamos tres elementos del tipo P con IDES. Los IDES son resultados y resultados 3. Y tenemos un código JavaScript dentro de las etiquetas de script en el que hemos declarado cuatro variables de ejemplo, tres variables de resultado y al final tenemos tres líneas que lo que nos hacen es introducir estos resultados dentro de los elementos definidos en el código HTML. Si analizamos las variables ejemplo 1 y ejemplo 2, veremos que son idénticas a ejemplo 3 y ejemplo 4. La única diferencia es que en el ejemplo 3 y el 4 las hemos añadido entre comillas. Este es un ejemplo práctico para demostrar lo que hemos comentado anteriormente. Si declaramos las variables y asignamos el valor numérico sin más, esto funcionará como número. Pero en el caso de que asignemos el valor introduciendo el número entre comillas, JavaScript interpretará que esto es una cadena de texto y trabajará con este valor como si de una cadena de texto se tratase. Si nos fijamos en los resultados en la suma del ejemplo 1 y el ejemplo 2, al ser dos variables numéricas que en ningún caso hay ninguna de ellas entre comillas, vemos que el resultado es 11. Vemos que efectivamente el código ejecutará a suma de los dos valores numéricos. Si analizamos el ejemplo 2 donde estamos sumando las variables 3 y 4, aquí como hemos comentado anteriormente, lo que se está produciendo es una concatenación de dos variables de tipo string, la concatenación de dos cadenas de texto. Por lo tanto, el resultado 2 que se muestra en pantalla es un 56, pero hay que tener en cuenta que para el código no es un 56 numérico. Es otra cadena de textos que si a este 56 ahora quisiéramos sumarle algo de la misma forma que lo hemos sumado este 5 y este 6 de tipo string, lo que haríamos sería seguir concatenando letras. Dicho de otra forma, si el ejemplo 4 en vez de ser un 6 fuese una, en el resultado 2 veríamos 5a, porque este 56 para el código es 100% texto, no es numérico. Y también analizamos el resultado 3. En resultado 3 lo que se pretende es demostrar lo que hemos dicho antes. Antes hemos dicho que cualquier tipo de variable sumada a una variable del tipo string trabaja o funciona como si fuese string. Entonces en este caso vemos que el ejemplo 2 es un 6 numérico y vemos que el ejemplo 4 es un 6, pero que no es numérico, que es de texto. Entonces lo que ha hecho el código es convertir el ejemplo 2 en texto y al igual que en el ejemplo anterior, que en el resultado 2, pues en el resultado 3 vemos un 66, vemos un 66 que JavaScript interpreta como un texto y no como un número. Bueno, pues habiéndose visto estos ejemplos, espero que haya quedado claro la diferencia entre variable numérica y variable string, veremos los otros tipos de variable que contiene este lenguaje de programación. Por un lado tenemos también las variables Boolean, que son variables que únicamente pueden almacenar dos valores, o verdadero o falso. En este tipo de variable la asignación del valor no la tenemos que hacer entre comillas, en este caso no usamos las comillas, porque precisamente tu or false son palabras reservadas del lenguaje, que el lenguaje ya sabe en qué contexto se utilizan y ya la sabe interpretar, por lo tanto las Boolean nunca, nunca se asignan entre comillas. Y por otro lado tenemos las variables de tipo array que permiten almacenar múltiples valores dentro de una misma variable. Aquí tenemos tres ejemplos de declaración de arrays, los tres son exactamente iguales, funcionarían igual a efectos prácticos. Estamos declarando una variable con el nombre de cars en la que estamos añadiendo tres marcas de coches, en la primera lo hacemos en una línea, en la segunda forma es exactamente igual lo único que estamos introduciendo saltos de línea, en este caso los saltos de línea no afectan a la declaración de la variable y se declaran exactamente igual. Y en el tercer caso estamos declarando una variable que es cars, la estamos declarando como array al introducir estos corchetes y posteriormente en otras líneas de código, en otras sentencias lo que estamos haciendo es añadirles específicamente en posiciones determinadas valores hasta variable. En este caso lo estamos haciendo en el mismo orden, estamos reinando la posición 0, 1 y 2 posiciones correlativas, pero podríamos decidir hacerlo diferente. Podríamos guardar cualquier valor en cualquier posición de la array, independientemente de que los anteriores valores estuvieran llenos o no. Y también bueno después una forma de acceder a estas variables, a estas posiciones pues es introducir cars corchetes, el número de índice al que queremos acceder y cerrar corchetes. De la misma forma que en el ejemplo 3 declaramos y llenamos estas posiciones luego podríamos usar esta forma para acceder a los valores y consultarlos o utilizarlos posteriormente en cualquier parte del código. Y por otro lado también podemos utilizar funciones específicas del lenguaje de programación que nos permiten cambiar valores de tipo de dato. Aquí tenemos cuatro ejemplos, puedo convertir cualquier cosa en número, puedo convertirlo en decimal, en entero y en string y en cadena de texto.
