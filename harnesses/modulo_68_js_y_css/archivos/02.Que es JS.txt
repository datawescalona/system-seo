Hola de nuevo, empezamos con el primer punto del curso de JavaScript en el que precisamente vamos a ver qué es este lenguaje y sus principales características. Bien, pues JavaScript es un lenguaje de programación que es utilizado para crear funciones en páginas webs, permite la actualización dinámica de contenidos, generar interacciones con el usuario, pedir información, mostrar datos, etcétera, etcétera. Y es importante destacar que es un lenguaje que se ejecuta en entorno cliente. Entendemos como cliente a cualquier programa que se conecta a internet para obtener cualquier tipo de información. Y bueno, más adelante veremos en el segundo punto la diferencia principal entre entorno cliente, entorno servidor y matizaremos mejor que significa que es ya un lenguaje de programación en entorno cliente.
