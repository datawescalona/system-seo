En el tercer punto de este curso, veremos los elementos semánticos de las páginas HTML. El lenguaje HTML contiene elementos semánticos y no semánticos. Los elementos no semánticos, etiquetas, son elementos que no dicen nada, que no ayudan a entender el contenido que hay dentro de estos elementos. Normalmente se utilizan para facilitar la programación del desarrollador o el desarrollo de los elementos. Son muy útiles para aplicar estilos y estructurar la página web, aparte de la estructura principal que comentábamos anteriormente, pues tener una estructura un poco diferente para aplicar estilos y otras tipo de interacciones. Ejemplos de estos elementos son las etiquetas div y span. Por otro lado, tenemos los elementos semánticos. Estos elementos definen un poco el contenido que contienen. Por ejemplo, la etiqueta form, definen que dentro de esta etiqueta hay un formulario, la etiqueta table o table que define, bueno, que va a ser una tabla, la etiqueta article, etc. En algunas páginas web podemos encontrar códigos HTML como los siguientes. Ahora explicaremos los elementos semánticos básicos del código HTML. Tenemos el elemento article, su contenido ha de ser específico, independiente y autónomo.
