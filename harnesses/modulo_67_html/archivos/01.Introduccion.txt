Pues bien, empezaremos con el primer punto, HTML. HTML básicamente es un lenguaje de programación que se utiliza para la creación de páginas web. La estructura y los contenidos de las páginas web están todos escritos en el lenguaje HTML. Hoy en día no es el único lenguaje que se utiliza para la creación de las páginas web, ¿vale? El lenguaje de CSS es, por ejemplo, marca la parte de estilos, hay otros lenguajes como PHP, JavaScript, que tienen sus funciones. Pero toda página web está escritos en HTML y la estructura de la misma también. HTML, como hemos comentado anteriormente, está formado por una serie de elementos y se considera un lenguaje de marcas. ¿Por qué es un lenguaje de marcas? Porque todos estos elementos se caracterizan por incluir los textos dentro de limitadores. Y gracias a estos limitadores, posteriormente los navegadores, otros lenguajes, otras aplicaciones pueden identificar estos elementos y leerlos o manipularlos. Si observamos este ejemplo de código de HTML, veremos básicamente que, bueno, las sintaxis de los elementos, ¿vale? Las etiquetas del elemento están incluidas dentro de los símbolos menor y mayor que. Después incluimos el contenido y finalmente cerramos el elemento con la misma etiqueta, pero incluyendo una barra de cierre después del símbolo menor que. Estos elementos, además, pueden contener atributos, ¿vale? La sintaxis para incluir estos atributos es básicamente, bueno, la misma sintaxis que la comentaba anteriormente, pero añadiendo el nombre de atributo, un igual y el valor entre comillas.
