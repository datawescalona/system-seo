Hola, bienvenidos al cuarto punto del curso de servidores del Master Deseo de Vixeo, en el que comentaremos otros servidores que son importantes para correcto funcionamiento de internet, pero los hemos metido todos en un mismo punto porque no dan para hablar tanto como el servidor remoto o el servidor DNS que hemos comentado en puntos anteriores. El primero de ellos es el servidor web, son todas aquellas aplicaciones que nos permiten conectarnos a páginas webs y nos saben contestar con los contenidos, imágenes y estilos que estamos solicitando. Estos servidores webs suelen trabajar por el puerto 80 para la versión HTTPM y con el puerto 443 para la versión securizada con HTTPS, aunque desde la configuración del servidor podemos cambiar estos parámetros y poner que se queden a las cuchas estos servidores en los puertos que queramos o que necesitemos. En este servidor, en los servidores webs se pueden configurar redirecciones y otras cosas como contraseñas para el acceso a ciertas carpetas. Es posible configurar virtualhost para que un mismo servidor web pueda alojar varias páginas webs. En un punto anterior lo hemos visto en el ejemplo de linkfp que había dos nombres de dominio apuntando en las DNS a la misma IP. Estos gracias a las configuraciones del tipo virtualhost que nos evitan tener que ir montando diferentes servidores webs y acabar teniendo un servidor web por página. Y algunos ejemplos de servidor web son POSAPACHE, Internet Information Server o ENGINX. Otro servidor muy común es el servidor de archivos FTP que se encarga de permitir la transferencia de archivos bidireccional. Tanto podemos descargarnos como subir archivos a este servidor utilizando el protocolo FTP. El protocolo FTP utiliza dos puertos, el 21 para la autenticación y el 20 para la transferencia de archivos. Y en su versión segura el SFTP utiliza el puerto 22. Dentro de este servidor FTP es posible utilizar los propios usuarios de sistema o crear usuarios nuevos y específicos únicamente con accesos FTP para poder delimitar permisos de lectura de escritura. Por otro lado tenemos también los servidores de bases de datos. En el siguiente curso entraremos en detalle sobre las bases de datos MySQL, pero bueno a nivel global estos servidores se encargan de organizar y recopilar información. Por defecto utilizan el puerto 3306 y comentar que es posible administrar una base de datos vía consola con aplicaciones clientes de bases de datos o con la aplicación PHP MyAdmin si esta está instalada en el servidor.
