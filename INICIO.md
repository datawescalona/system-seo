# INICIO — SISTEMA HARNESS-SDD
# Compatible con: Claude, ChatGPT, Gemini, cualquier LLM

Hola. Este es mi sistema de construcción de arneses SEO. Estoy
construyendo el motor (Sistema 1). Todavía no hay clientes.

Por favor ayúdame así:

0. Si tienes acceso a terminal y git en esta carpeta (ej. Claude Code):
   sigue PRIMERO el "ARRANQUE BLINDADO" de CLAUDE.md (fetch + reset al
   repo en vivo, reinstalar el hook si falta, correr check_coherencia.sh).
   Si ese chequeo falla, avísame y no sigas. Si no tienes terminal/git
   (ej. chat sin herramientas), al menos usa siempre la URL raw de abajo,
   nunca una copia pegada de una sesión vieja.

1. Lee mi archivo de estado:
   https://raw.githubusercontent.com/datawescalona/system-seo/main/CONTROL.md

2. Dime en lenguaje simple por dónde voy (qué módulo, cuántos arneses
   llevo) y cuál es mi "PRÓXIMA TAREA". Esto solo se dice DESPUÉS de
   pasar el chequeo del paso 0 cuando aplique.

3. Propón el siguiente paso y pregúntame si lo hacemos.

Si quiero construir el arnés de un módulo, lee los archivos reales de
su carpeta (harnesses/modulo_XX/archivos/) y arma el ARNES.md con la
plantilla del HARNESS_SDD_BLUEPRINT.md. Nunca inventes el contenido:
si no puedes leer los archivos, pídemelos.

El CONTROL.md manda sobre dónde voy. No me preguntes cosas que ya
están en el repo.
