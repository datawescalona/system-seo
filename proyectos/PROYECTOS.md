## 🧠 INSTRUCCIÓN PARA CUALQUIER IA
Este archivo es independiente de CONTROL.md (que rastrea la
construcción de los arneses, módulo por módulo).
Aquí se rastrea el trabajo real con clientes o proyectos,
usando los arneses ya construidos como herramientas.
Lee este archivo antes de iniciar o continuar cualquier proyecto.
No mezcles el avance de esta capa con el avance de CONTROL.md —
son dos sistemas paralelos e independientes.

---

## 📌 CÓMO FUNCIONA ESTA CAPA
- Cada proyecto tiene su propia carpeta dentro de /proyectos/
- Cada carpeta de proyecto contiene un archivo PROYECTO.md,
  basado en la plantilla de proyectos/_PLANTILLA/PROYECTO.md
- Este archivo (PROYECTOS.md) es solo el índice maestro: lista
  todos los proyectos, su estado, y etiquetas para relacionarlos
  entre sí (saber qué es parecido a qué)

## 🏷️ SISTEMA DE ETIQUETAS
Usa etiquetas consistentes en cada proyecto para poder buscar
"proyectos parecidos" más adelante y reusar criterios:

- Tipo de negocio: ecommerce, servicios B2B, servicios B2C, blog,
  sitio informacional, local, internacional
- Tamaño del proyecto: pequeño, mediano, grande
- Fase de trabajo: keyword research, arquitectura, on-page,
  auditoría, migración, etc.

## 🔁 CÓMO ACTIVAR UN PROYECTO EXISTENTE EN CUALQUIER IA
0. Si el Mega Arnés (Orquestador) ya existe, actívalo primero
   pegando MEGA_ARNES.md — él decide qué arneses usar y en qué
   orden, y consulta el PROYECTO.md para no repetir trabajo
1. Pega el contenido de este archivo (PROYECTOS.md)
2. Pega el contenido de proyectos/[nombre-del-proyecto]/PROYECTO.md
3. Adjunta los ARNES.md (y sus archivos) de los módulos que ese
   proyecto necesite — según indique el Orquestador o el campo
   "PRÓXIMA TAREA"
4. Continúa desde donde indica el campo "PRÓXIMA TAREA" de ese
   proyecto específico

---

## 🔁 CÓMO EMPEZAR UN PROYECTO NUEVO
1. Crea una carpeta nueva dentro de /proyectos/ con el nombre
   del cliente o proyecto (ej: proyectos/cliente-tienda-sofas/)
2. Copia la plantilla de proyectos/_PLANTILLA/PROYECTO.md ahí
   y complétala
3. Agrega una fila nueva en el índice de arriba (📋 ÍNDICE DE
   PROYECTOS) con sus etiquetas correspondientes
